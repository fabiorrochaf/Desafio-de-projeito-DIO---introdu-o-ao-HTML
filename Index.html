O que é HTML e para que ele serve?

(voltar ao inicio)

Criado por Tim Berners-Lee, HTML (linguagem de marcação de Hipertexto), é o componente básico da web, permite inserir o conteúdo e estabelecer a estrutura básica de um website. Serve para dar significado e organizar informações de uma página web.HTML não é uma linguagem de programação, por mais que tenha Linguagem no nome, é basicamente um arquivo de marcação, com delimitadores que são interpretados pelo navegador.

"O objetivo da Web é servir a humanidade. Nós construí-lo agora para que aqueles que venham mais tarde possam criar coisas que não podemos imaginar."
-Tim Berners-Lee

Doctype - Definindo o documento

            
O doctype não é uma tag HTML, mas uma instrução para o navegador e outros programas que podem ler seu site, que o código encontrado ali é um código HTML. Assim eles sabem o que fazer para mostrar seu site da melhor forma possível

            
HTML

            
 Abertura e fechamento do HTML, tudo que estiver entre eles o computador irá ler como tal

            
HEAD

            
"cabeça", informações pré-carregadas antes de exibir para o usuário, são dados implícitos, de uso e controle do documento: vinculação com outros arquivos, aplicação de lógica de programação de scripts e metadados. Todo o conteúdo do cabeçalho fica delimitado entre a abertura e fechamento da tag < head > e < /head >.

            
BODY

            
tudo que o usuário irá de fato ver e navegar, todo e qualquer texto ou mídia. fica delimitado entre a abertura e fechamen to da tag < body > e < /body >.

            
Tags e seus atributos
 (voltar ao inicio)
            
Tags são o conjunto de caracteres que formam um elemento. Existem dois tipos de Tags, as que necessitam de fechamento e as que não necessitam de fechamento.

            
As que necessitam de fechamento, utilizamos o sinal de menor ( < ), seguido do nome do elemento e o sinal de maior ( > ) para abertura. Para fechamento, utilizamos o sinal de menor ( < ), seguido de barra ( / ), nome do elemento e o sinal de maior ( > ).

            
Exemplo:

            
 < p >A tag do elemento de parágrafo necessita de fechamento.< /p > 

            
Já os atributos são informações que passamos na Tag para que ela se comporte da maneira esperada. Existem atributos globais (que funcionam em todas as Tags) e específicos (que são direcionados para cada Tag, através de especificação).

            
exemplo:

            
< input type="text" ou "number" />

            
Nesse caso a Tag input abriria um campo texto no HTML. Com o atributo type você consegue escolher por exemplo se no campo texto você só colocaria texto ou números
 
            
            
Listas ordenadas e não ordenadas
 (voltar ao inicio)
            
A principal diferença entre a lista ordenada e a lista não-ordenada está na maneira com que os itens de cada lista são apresentados - a lista ordenada enumera os elementos; a lista não-ordenada coloca marcadores antes de cada elemento.

            
O uso de números induz o leitor a esperar uma seqüência coerentemente ordenada de itens. O uso de marcadores induz o leitor a esperar uma coleção de itens (que não precisa ter necessariamente uma ordem).

            
Exemplo Lista ordenada:

            
< ol >
                                    
            
< li >item 1< /li>

            
< li >item 2< /li>

            
< li >item 3< /li>

            
< /ol >

            
 Resultado

            
    
                
  1. item 1
    2. 
                
  2. item 2
    3. 
                
  3. item 3
    4. 
            

            
Uma lista ordenada é delimitada pelas etiquetas < OL> e < /OL>  Cada item da lista é delimitado pelas etiquetas < LI> e < /LI>

            
exemplo Lista não ordenada:

            
< ul>

            
< li>item 1< /li>

            
< li>item 2< /li>

            
< li>item 3< /li>

            
< /ul>

            
Resultado:

            
    
                
  • item 1
    • 
                
  • item 2
    • 
                
  • item 3
    • 
            

            
A estrutura da lista não-ordenada é semelhante a da lista ordenada - porém, ao invés de ser construída com o elemento OL, é construída com o elemento UL

            
Links
 (voltar ao inicio)
            
O elemento < a> em HTML (ou elemento âncora), com o atributo href cria-se um hiperligação nas páginas web, arquivos, endereços de emails, ligações na mesma página ou endereços na URL. O conteúdo dentro de cada < a> precisará indicar o destino do link.
